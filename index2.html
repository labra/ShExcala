ShExcala - Shape Expressions in Scala

Introduction

ShExcala is an implementation of ShEx (Shape Expression) language that can be used to define shapes of RDF graphs.

  • Source code: Github project
  • Latest release (binary jar): Jars and Javadocs
  • Packaged binaries: binaries/shexcala-0.1.8.zip, binaries/shexcala-0.1.8.tgz
  • Documentation: Wiki
