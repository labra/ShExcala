Shape Expressions Implementation

Introduction

ShExcala is an implementation of the Shape Expressions language that can be used to describe and validate RDF graphs.

ShExcala is implemented in Scala which means that it can be used in any JVM environment.

We are currently working on adapting the implementation to ScalaJS so it could also be run within Javascript environments

ShExcala can be used as an independent command line tool or as a jar library that can be invoked from Scala or Java programs.

It is being used as the basis of the RDFShape online validator tool

More information is available at ShExcala Wiki pages.
